# Maaza Mango Beverage Website

A premium, ultra‑realistic mango‑flavored beverage showcase with video hero, dynamic product galleries, and a simple order backend.

## Live Demo

🚀 **Netlify deployment:** https://charming-melomakarona-645258.netlify.app

## Repository

Source code: https://github.com/susmithareddy3112/maaza-mango

## Features

- Vibrant premium visual design (glass bottle, can, video hero)
- Multiple style variations (luxury, fun Gen‑Z, natural, frozen)
- Order form backed by a local JSON‑based database
- Responsive layout, glass‑morphism, smooth animations

## How to Run Locally

```bash
npm install
node server.js   # runs an Express server on http://localhost:8080
```
